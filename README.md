# Helix

多功能浏览器工具集（Chrome 扩展程序）。

## 功能

### 二维码生成器

支持长文本分段生成二维码（二维码链），搭配 [Echo](https://github.com/blue-lemon0/Echo) Android App 的连续扫码模式实现自动解析拼接。

- **分段二维码**：超长文本按字节切分，每段生成独立二维码
- **密度调节**：滑块控制每段字节数（200~2000），拖拽即重新生成
- **分页浏览**：多张二维码翻页查看
- **自动播放**：类媒体播放器控制，自动循环翻页，速度可调（1s~5s）
- **批量下载**：下载当前或全部二维码

### JSON 构造器

可视化字段树编辑器，所见即所得生成 JSON。

- **字段树编辑**：添加/删除/修改字段，支持嵌套对象和数组
- **类型一键切换**：V（字符串）↔ {}（对象）↔ []（数组）
- **字面量支持**：引号开关控制 null、true、false、数字无需引号
- **实时预览**：语法高亮 JSON 预览
- **导入/导出**：从 JSON 导入生成字段树，复制到剪贴板
- **内联编辑**：直接编辑 JSON 文本后同步回字段树
- **重名检测**：同级别字段重名时橙色边框警告
- **键盘导航**：Enter 在字段间跳转，自动新增字段

## 安装

1. 打开 Chrome，进入 `chrome://extensions/`
2. 开启「开发者模式」
3. 点击「加载已解压的扩展程序」，选择本项目根目录

## 使用

点击扩展图标 → 二维码生成器 → 输入文本 → 生成二维码

### 配合 Echo 使用

1. 在 Echo App 中切换到「连续」扫码模式
2. 在 Helix 中生成链式二维码，点击「自动播放」
3. 用 Echo 依次扫描屏幕上逐帧显示的二维码
4. 扫描完成后 Echo 自动拼接还原完整文本

## 二维码链格式

每段二维码内容为 JSON 包，[Echo](https://github.com/blue-lemon0/Echo) 按此协议自动解析拼接：

```json
{"v":1,"t":3,"i":0,"d":"文本段"}
```

- `v`: 版本（当前 1）
- `t`: 总段数
- `i`: 当前段索引（从 0 起）
- `d`: 文本数据
