# Helix

多功能浏览器工具集（Chrome 扩展程序）。

## 功能

### 二维码生成器

支持长文本分段生成二维码（二维码链），搭配 App 端连续扫码实现自动解析拼接。

- **分段二维码**：超长文本按字节切分，每段生成独立二维码
- **密度调节**：滑块控制每段字节数（200~2000），拖拽即重新生成
- **分页浏览**：多张二维码翻页查看
- **自动播放**：类媒体播放器控制，自动循环翻页，速度可调（1s~5s）
- **批量下载**：下载当前或全部二维码

## 安装

1. 打开 Chrome，进入 `chrome://extensions/`
2. 开启「开发者模式」
3. 点击「加载已解压的扩展程序」，选择本项目根目录

## 使用

点击扩展图标 → 二维码生成器 → 输入文本 → 生成二维码

## 二维码链格式

每段二维码内容为 JSON 包：

```json
{"v":1,"t":3,"i":0,"d":"文本段"}
```

- `v`: 版本（当前 1）
- `t`: 总段数
- `i`: 当前段索引（从 0 起）
- `d`: 文本数据
